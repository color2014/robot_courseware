    三、异常#

    什么是异常?

    程序在运行过程中，发生了未知的事件，影响到了程序的正常运行 异常是一种事件 异常会影响到程序正常运行

    1. 第一个异常#

    1
2
3
4
    a = 10
b = 0
c = a/b
print(c)

    运行程序:

    出现了ZeroDivisionError异常

    出现异常会造成程序停止

    3. finally#

    try语句中使用finally,finally中代码不论有没有出现异常都会执行

    格式:

    1
2
3
4
    try:
    逻辑代码
finally:
    无论是否出现异常，都会执行

    1
2
3
4
    try:
    a = 1 / 0
finally:
    print(“hello”)

    程序出现异常也会执行finally中的代码

    7. 自定义异常#

    除了系统已经定义好的异常,也可以自己定义异常

    异常的继承关系

    所有的异常都继承自Exception

    自定义异常的格式:

2
3
4

    class 自定义异常名(Exception):

    def __init__(self, msg):
        self.msg = msg

    def __str__(self):
         return self.msg

    代码:

    class MyException(Exception):
    def __init__(self,msg):
        '''
        异常的提示文字
        :param msg:
        '''
        self.msg = msg

    def __str__(self):
        return self.msg

a = 10
b = 0
try:
    if b==0:
        raise MyException('出现了0')
    result = a/b
    print(result)
except MyException as error:
    print('捕获异常',error)

    raise的作用是抛出异常

    抛出异常如果没有通过try except处理,程序会停止
